Background


On one of the rabbit holes of the internet I run into the Virtuscope, a 3D printed cyberdeck. I really liked the idea and impulsively I started to work on recreating it. Soon after starting I realized that I was lacking many of the tools and skills necessary to build something like it.

My first attempt at building and modding it is documented in a blog post I posted in the cyberdeck café site, The main findings were that a purely raspberry pi-based system wouldn’t be able to run the software I wanted for a viable daily driver computer and that to fit the hardware I had in mind, there would need to be a significant number of design changes.

Build

The current build includes an i7 NUC on the left and a raspberry pi 4 on the right. The raspberry pi is able to run on the go thanks to a couple of 3.6mAh 18650.

The front has the keyboard and under it there is a KVM switch that allows the msg to share the screen, keyboard and a usb plug between both systems, for a seamless transition.

The screen is a 7inch touchscreen, leaving some space to the right of it that I used to add an eInk display, breakout board and a Pimoroni micro dot phat.
